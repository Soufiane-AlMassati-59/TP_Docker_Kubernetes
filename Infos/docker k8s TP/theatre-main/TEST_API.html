🎯 Endpoints Principaux

POST Inscription

http://localhost:8080/api/auth/register

Body JSON :

{
  "nom": "Dupont",
  "prenom": "Jean",
  "email": "jean.dupont@example.com",
  "motDePasse": "SecurePass123!"
}

POST Connexion

http://localhost:8080/api/auth/login

Body JSON :

{
  "email": "jean.dupont@example.com",
  "motDePasse": "SecurePass123!"
}

Retourne un token JWT à utiliser dans les requêtes suivantes

GET Liste des Spectacles

http://localhost:8080/api/spectacles

Récupère tous les spectacles disponibles

GET Détails d'un Spectacle

http://localhost:8080/api/spectacles/{id}

Exemple : http://localhost:8080/api/spectacles/1

GET Spectacles Populaires

http://localhost:8080/api/spectacles/populaires

Top 5 des spectacles les plus réservés

POST Créer une Réservation

http://localhost:8080/api/reservations

Headers :

Authorization: Bearer {votre_token_jwt}

Body JSON :

{
  "idSpectacle": 1,
  "nombrePlaces": 2
}

GET Mes Réservations

http://localhost:8080/api/reservations/mes-reservations

Headers :

Authorization: Bearer {votre_token_jwt}

GET Statistiques

http://localhost:8080/api/statistiques/dashboard

Vue d'ensemble des statistiques de la plateforme
